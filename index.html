자동 중복 처리 방식
C열(팀/유통망) + E열(이름)이 동일하면 같은 사람으로 판단 → A열(타임스탬프) 기준 최신 응답 1건만 반영합니다.
재신청 시 이름 옆에 재응답 배지가 표시됩니다.
⚠ 요약 표는 별도 시트에 작성하세요
구글 폼 응답 시트에 요약 표를 함께 넣으면 CSV에 불필요한 행이 포함되어 데이터가 꼬입니다. 구글 시트에서 시트 추가(+)를 눌러 새 시트(예: "요약")에 따로 작성하세요.
새 교육 추가
교육 이름과 구글 시트 CSV URL을 입력하면 탭으로 관리됩니다.
URL은 한 번 등록하면 브라우저에 저장되어 다음에 열어도 유지됩니다.
등록된 교육 목록
