자동 중복 처리 방식

예시) 김민준 (010-1234-5678)
  3월 1일 09:00 → 참석  ← 이전 응답, 자동 제외
  3월 15일 10:00 → 불참  ← 최신 응답, 반영됨
구글 시트 CSV 연동
구글 폼 응답이 저장되는 스프레드시트를 웹에 게시하면, URL만 입력해서 데이터를 불러올 수 있어요.
1
구글 폼 응답 스프레드시트를 열어요.
2
상단 메뉴 파일 → 공유 → 웹에 게시를 클릭해요.
3
시트 선택 후 형식을 쉼표로 구분된 값(.csv)으로 설정 후 게시해요.
4
생성된 URL을 아래에 붙여넣고 불러오기를 눌러요.
https://docs.google.com/spreadsheets/d/[시트ID]/pub?gid=0&single=true&output=csv
열(컬럼) 매핑
구글 폼 응답 시트의 열 순서에 맞게 설정하세요. 구글 폼은 자동으로 1열에 타임스탬프를 기록합니다.
중복 판단 기준: 이름 + 연락처가 동일 → 타임스탬프 최신 응답만 유지
